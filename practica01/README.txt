Practica 01 de Lógica computacional
La practica01 se puede abrir de dos formas:
    1. En la terminal y en la carpeta \practica01, poner "ghci *.hs"
    2. EN la terminal y en la carpeta \practica01, poner "ghci", y una vez dentro, poner ":l Practica01.hs" 
El trabajo se diseñó con 2 incisos del puntos extra:
    1. Se trabajó con módulos, exactamente tres: 
        -Practica01 como el módulo pricipal
        -Valuaciones donde se define el tipo de las valuaciones y los modelos
        -Variables donde se define el tipo de las variables y fórmulas de PL
    2. Se incluyó una lista de pruebas debajo de cada función en forma de comentarios, solo se necesita copiar y pegar en la terminal para correr las pruebas.
